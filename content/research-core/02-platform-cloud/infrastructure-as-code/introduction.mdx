## Research focus

Infrastructure as Code (IaC) enables **repeatable, auditable** cloud provisioning. This track covers module design, remote state, workspace strategy, and security scanning before `terraform apply`.

### Core questions

1. How should modules be versioned and consumed across teams?
2. What state backend layout supports disaster recovery and least privilege?
3. Which policies must block apply vs warn in CI?
