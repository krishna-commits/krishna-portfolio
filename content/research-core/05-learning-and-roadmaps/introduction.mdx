# Learning & Roadmaps

This pillar is a **self-contained curriculum** for engineers growing into DevSecOps and platform roles. Everything here is written for this Research Core — not a link dump to third-party awesome lists.

## What you will find

| Track | Purpose |
|-------|---------|
| **DevOps Learning Hub** | Phased roadmaps from Git basics to production DevSecOps |
| Tool guides | What to learn, in what order, and which Research Core articles to read next |
| Certification map | AWS, Azure, GCP, and Kubernetes cert sequencing |
| Interview prep | Topics and scenarios for senior platform interviews |

## How to use it

1. **New to DevOps?** Start with *DevOps Foundation Roadmap* in the Learning Hub.
2. **8+ years in production?** Use *Senior Platform Gap Checklist* and security chapters.
3. **Studying for exams?** Open *Certification Roadmap* and match to your cloud choice.
4. **Deep dives** — every step links to hands-on docs in other Research Core pillars.

## Author note

These guides reflect **8+ years** securing and automating cloud platforms across AWS, GCP, Azure, and Kubernetes — distilled into learnable sequences you can follow at your own pace.
