## Objective

Reliable systems require **visibility**. This track documents logging, metrics, and tracing patterns that support both performance tuning and security investigations.

### Tools studied

- Prometheus + Alertmanager + Grafana
- Elastic Stack (Elasticsearch, Kibana, Fluentd)
- Nagios and legacy monitoring integration
- Kafka as log/event transport
