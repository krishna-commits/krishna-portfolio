# Operations & Reliability

Operations research connects **day-two engineering** — keeping systems fast, available, and observable — with security incident detection and capacity planning.

## Research tracks

| Track | Focus |
|-------|--------|
| Linux | Kernel, storage, SELinux, LDAP, cron, commands |
| Networking | Protocols, firewalls, load balancing, OSPF |
| Database | MySQL, PostgreSQL, MongoDB operations |
| Logging and Monitoring | ELK, Prometheus, Grafana, Nagios, Kafka |
| SRE and Reliability Engineering | SLOs, error budgets, chaos engineering |

## Reliability ↔ security

Shared telemetry supports both performance tuning and **threat hunting**. This pillar documents dual-use observability patterns.
