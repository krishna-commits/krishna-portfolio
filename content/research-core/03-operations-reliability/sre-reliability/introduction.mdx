## Connection to security

Reliability and security overlap: **availability incidents** often follow compromises; weak observability delays breach detection. This track documents SRE practices that strengthen both.

### Topics

- Defining SLIs/SLOs for critical user journeys
- Error budget policies and release gates
- Chaos experiments for resilience validation
- On-call hygiene and runbook automation
