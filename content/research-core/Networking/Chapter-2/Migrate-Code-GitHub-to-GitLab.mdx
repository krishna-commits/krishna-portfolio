_Migrate Code from GitHub to GitLab_

For this, I am using **“Project”** as an example, directory or repository names vary depending on the one you are using

1. Your project directory

```
cd Project/
```

2. Check the remote origin values, which are GitHub URLs.

```
git remote -v
```

3. Fetch all of the remote branches and tags:

```
git fetch origin
```

4. Update remote URL from GitHub to GitLab

```
git remote set-url origin git@gitlab.com:krishna-commits/Project.git
```

5. Push all to the origin, i.e. GitLab URL

```
git push origin --all
```

6. Final checks

```
git remote -v

git branch -a
```
