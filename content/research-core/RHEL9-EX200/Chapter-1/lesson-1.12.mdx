# Using Wildcard Characters

- ? --> Matches any single character
- `*` --> Matches any number of character
- [a-z] --> Matches within the range
- [aZ] --> Matches within the set 
- [^a-z] --> Matches out of the range
- [^az] --> Matches out of the set
