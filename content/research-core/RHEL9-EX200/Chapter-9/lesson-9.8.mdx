## Not Important for examination

### Viewing SELinux Boolean Status 
```bash 
[root@server ~]# getsebool -a 
abrt_anon_write --> off
abrt_handle_event --> off
abrt_upload_watch_anon_write --> on
...
```

### Enable or disable SELinux Boolean parameter
```bash 
[root@server ~]# setsebool -P ftpd_full_access=1 
[root@server ~]# getsebool -a | grep ftpd_full_access 
ftpd_full_access --> on

[root@server ~]# setsebool -P ftpd_full_access=0
[root@server ~]# getsebool -a | grep ftpd_full_access 
ftpd_full_access --> off

```
