### Format

```
Min  Hour Day  Mon  Weekday
```
```
*    *    *    *    *  command to be executed

┬    ┬    ┬    ┬    ┬
│    │    │    │    └─  Weekday  (0=Sun .. 6=Sat)
│    │    │    └──────  Month    (1..12)
│    │    └───────────  Day      (1..31)
│    └────────────────  Hour     (0..23)
└─────────────────────  Minute   (0..59)
```

## Crontab


### Adding tasks easily
```bash
echo "@reboot echo hi" | crontab
```
### Open in editor - optional for another user
```bash
crontab -e [-u user]
```
### List tasks - optional for another user
```bash
crontab -l [-u user]
```
### Delete crontab file - optional for another user
```bash
crontab -r [-u user]
```
